# Geonwoo Kim — academic website

A single-page, content-first academic website inspired by the editorial restraint of Zoë Hitzig's personal site, without copying its text or layout verbatim.

## What is intentionally absent

- top navigation
- About and Contact sections
- footer and “last updated” text
- cards, badges, gradients, shadows, icons, and paper figures
- redundant per-paper labels

## Portrait behavior

The site expects a real portrait at:

```text
assets/portrait.jpg
```

No fake portrait is included. Until that file is added, the page automatically collapses to a polished text-only introduction. Use a vertical image, ideally 4:5 or slightly taller, at least 1200 px wide.
