# Geonwoo Kim academic website — v6

A restrained editorial academic site inspired by the composition and typography of Zoë Hitzig’s personal site, with a small spectral/matrix visual as the personal accent.

## Changes from v5

- Refined spectral study with more natural depth, restrained cobalt/oxblood palette, and subtler matrix structure.
- Reduced title weight and size.
- Improved hero proportions and text measure.
- Reduced the gap before Research.
- Made the Research hierarchy quieter and more consistent.
- Removed redundant paper-status lines.
- Removed the figure caption.
- Removed JavaScript entirely.
